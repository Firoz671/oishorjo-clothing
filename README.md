# Oishorjo Clothing (ঐশ্বর্য)

A production-ready frontend for a premium South Asian luxury streetwear brand. Built with a focus on editorial aesthetics, cinematic layouts, and modern web performance.

## Tech Stack

- **React.js** (Vite)
- **Tailwind CSS** (Custom theme & typography)
- **Framer Motion** (Page transitions & micro-animations)
- **React Router v6**
- **Lucide React** (Iconography)

## Features

- 🌓 **Editorial Design**: Dark-themed, cinematic UI with custom typography (Playfair Display, Space Mono).
- 🛒 **Cart Management**: Fully functional shopping cart utilizing React Context API and `localStorage`.
- 📱 **Responsive**: Flawless layout scaling from mobile screens to ultra-wide desktop monitors.
- ⚡ **Performant**: Lightning-fast Vite dev server and optimized React functional components.

## Getting Started

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Run the development server**
   ```bash
   npm run dev
   ```

3. **Build for production**
   ```bash
   npm run build
   ```
